Common Issues

Issue #1: Proxy is blocked:

  1. The proxy being blocked is a common thing and is quite easy to fix. If you saw that our proxy website is blocked, then please report the proxy block here! We frequently update the servers and create new ones. Check for other web proxies on our website!

Issue #2: Slow proxy traffic:

  1. If you see that the website loading speed is slow while using our proxy, it might be that a lot of people are using the proxy. To fix this, go to other web proxies on our website and select the one that has a better loading speed!

Issue #3: Cloudflare error:

  1. When surfing the web with the proxy, you might notice that some websites get a cloudflare error “This web property is not accessible via this address”. This is because the website you got this error on is using the DDoS cloudflare protection. Cloudflare detects our proxy and stops you from accessing the website. There is sadly no way to fix this.

Issue #4: Deceptive page error:

  1. Google might warn you that our proxy page is deceptive (malware/viruses) at some point. This is because every page a user visits is saved locally on the server. Google might detect a certain page someone has visited using the proxy and warn you. Just don’t visit that page and you’ll be fine.

If you have any issues that are not listed on here, please contant support on our discord server!
